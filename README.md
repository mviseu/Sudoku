# Sudoku
C++ implementation of the Sudoku game

![alt tag](https://raw.githubusercontent.com/mviseu/Sudoku/master/screenshots/Sudoku.png)

## Build
```
git clone git@github.com:mviseu/Sudoku.git
mkdir build
cd build
cmake -G "Unix Makefiles" ..
cmake --build .
```

## Run
```
./sudoku
```
